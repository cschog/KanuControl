# KanuControl V0.9

Veröffentlichung: 09.06.2026

## ✨ Teilnehmerverwaltung

### Einfachere Auswahl von Teilnehmern
Die Teilnehmerauswahl wurde verbessert:

- Ganze Zeilen können jetzt angeklickt werden.
- Die Auswahl wird weiterhin über die Checkbox angezeigt.
- Besonders auf Tablets und Smartphones deutlich einfacher zu bedienen.

### Schutz vor versehentlichem Löschen
Teilnehmer können nicht mehr entfernt werden, wenn sie in einer Reisekostenabrechnung verwendet werden:

- als Fahrer
- als Mitfahrer

Zunächst müssen die entsprechenden Einträge in der Reisekostenabrechnung entfernt werden.

### Bessere Fehlermeldungen
Fachliche Fehler werden jetzt direkt im Programm angezeigt und verständlich erklärt.

---

## 🚗 Reisekostenabrechnung

### Verbesserte Datenkonsistenz
KanuControl verhindert nun automatisch, dass Teilnehmer entfernt werden, die noch in einer Reisekostenabrechnung referenziert werden.

Dadurch werden inkonsistente Datenbestände vermieden.

---

## 👤 Personenverwaltung

### Automatische Erkennung der Bank
Wenn eine BIC angegeben wird und das Bankfeld leer ist, kann KanuControl den Banknamen automatisch ergänzen.

Bereits eingetragene Banknamen werden dabei niemals überschrieben.

### BIC-Datenbank
Für die automatische Bankermittlung wurde eine erste BIC-Datenbank integriert.

---

## 🔧 Technische Verbesserungen

- Zentralisierte Validierungen im Backend
- Verbesserte Fehlerbehandlung im Frontend
- Erweiterbare BIC-Datenbank auf Basis von CSV-Dateien
- Diverse Optimierungen der Benutzeroberfläche
